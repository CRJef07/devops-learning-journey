# Linux & DevOps Learning Journey

Personal knowledge base documenting hands-on learning in Linux Administration, DevOps,
and Cloud Engineering — built around real tools, real workflows, and production-grade
practices.

## Focus Areas

| Area | Topics |
| --- | --- |
| **Linux Administration** | File system, permissions, processes, systemd, networking, storage |
| **Bash Scripting** | Automation scripts, functions, error handling, cron |
| **Networking** | TCP/IP, DNS, firewalls, SSH, troubleshooting |
| **Git & GitHub** | Workflows, branching strategies, collaboration |
| **Docker** | Images, containers, volumes, Compose, networking |
| **Kubernetes** | Pods, deployments, services, RBAC, Helm |
| **AWS** | EC2, S3, IAM, VPC, CLI |
| **DevOps & SRE** | CI/CD, observability, incident response, IaC |
